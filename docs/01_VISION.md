# Visión de DINPRO

## La plataforma

DINPRO será una plataforma modular profesional para ingeniería lineal. No un programa. Una plataforma.

## Principios rectores

- **Plugin-based**: el Core no conoce los módulos. Los módulos conocen el Core.
- **Datos oficiales**: toda la información geoespacial y administrativa provendrá de fuentes oficiales.
- **Eterno**: diseñado para durar décadas. El Core es estable y no cambia.
- **Extensible**: cualquier persona podrá desarrollar un plugin sin tocar el núcleo.

## Ecosistema futuro

```
DINPRO
├── Plugin Topografía
├── Plugin LIDAR
├── Plugin BIM
├── Plugin IA
├── Plugin Catastro Francia
├── Plugin Portugal
└── ...
```

Todos funcionarán sobre el mismo Core. Todos serán independientes. Ninguno modificará el núcleo.

## Objetivo a largo plazo

Convertir DINPRO en el estándar de facto para el análisis, diseño y documentación técnica de infraestructuras lineales en el mundo hispanohablante.
